# QuizUp — Aplicación Móvil de Trivia

Aplicación móvil multiplataforma desarrollada con Flutter que permite a los usuarios participar en juegos de trivia con múltiples categorías y niveles de dificultad.

Incluye autenticación, ranking y persistencia en la nube.

## Stack

- Flutter
- Dart
- Supabase
- GoRouter
- HTTP
- Reactive Forms

## Funcionalidades

- Autenticación con Supabase
- Sistema de ranking
- Persistencia de récords
- Múltiples categorías
- Diferentes niveles de dificultad
- Arquitectura móvil multiplataforma
